* 自动校验
  * 基于双向绑定的解决方案， 比如 iview 和 element-ui
  * ant 表单校验，非双向绑定
* 基于双向绑定
  * 存在的问题，可能有需要 copy 原数据的场景，比如提交表单成功后才更新其他组件
  * data + rule => form，如果其他组件需要提交成功后才展示，就会表现成这样 cloneData + rule + data => [form, other]
* ant
  * data + rule，提供给 form 的只是初始值，data 直接提供给 other
  * form.getFieldsValue()
